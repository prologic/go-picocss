go-picocss is a classical "Hello World!" style Web App written in
[Go](https://golang.org) (*Golang*) and using the following frontend libraries:

- [PicoCSS](http://picocss.com/)

![Screenshot](./screenshot.png)
![Screenshot 2](./screenshot2.png)

## Installation

### Source

```#!bash
$ go get github.com/prologic/go-picocss
```

## Usage

Run go-picocss:

```#!bash
$ go-picocss 
```

Then visit: http://localhost:8000/
