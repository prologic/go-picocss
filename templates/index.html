Hello World!

Welcome to go-picocss a classical "Hello World" style Web App written in Go. using PicoCSS.

This is an example of using a "classless" CSS library with a Go backend server. It is completely responsive and will seamlessly resize on smaller devices. There is even a Hamburger style navication bar that will collapse automatically!

To try this out: 

      
        $ docker pull prologic/go-picocss
        $ docker run -p 8000:8000 prologic/go-picocss
